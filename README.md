# T20 Digit-Cricket Simulator & Tournament Engine

A robust frontend mathematical cricket simulation application featuring full franchise creation, custom player attribution mechanics, and dynamic tournament bracket lifecycle management.

[👉 Click here to play the live game!](PASTE_YOUR_GITHUB_PAGES_URL_HERE)

## 🛠️ Tech Stack & Architecture
- **Language:** Vanilla JavaScript (ES6+), HTML5, CSS3
- **Styling Framework:** Tailwind CSS
- **Data Persistence:** Browser `localStorage` API for cross-session saves
- **State Management:** Algorithmic structural handling of ball-by-ball, innings, and bracket progression

## 📈 Key Features
- **Deterministic Action Engine:** Translates sequence streams of digits into realistic cricket match data points (wickets, runs, rotation of strike) using custom mapping algorithms.
- **Roster & Attribute Customization:** Allows user creation of distinct teams and players with performance-altering traits (e.g., Aggressive, Defensive, Mystery Spin).
- **Automated Tournament Scheduler:** Formulates structural knockout brackets dynamically for 4, 6, or 8 teams.
